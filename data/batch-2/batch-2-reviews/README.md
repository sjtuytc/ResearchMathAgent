# Batch 2 Reviews

Anonymized referee reports for the First Proof second-batch submissions, organized as
`problem-NN/submission-X/`. Each submission folder contains the individual reviewer
source files (`submission-X-reviewer-N.tex`) and a combined PDF of all its reviews
(`submission-NX-reviews.pdf`).

## ⚠️ Download the PDFs — don't read them in GitHub's preview

**GitHub's in-browser PDF preview renders these documents incorrectly** — it substitutes
the wrong fonts (math comes out in sans-serif, and some symbols show as empty boxes). This
is a limitation of the older PDF.js viewer GitHub embeds, **not** a problem with the PDFs
themselves: they render correctly in every desktop PDF viewer, in `evince`/Preview/Acrobat,
and in current browsers' built-in viewers.

To read a review correctly, **download the PDF** and open it locally:

- On the file's GitHub page, click the **Download raw file** button (top right of the file
  view), or
- append `?raw=1` to the file URL, or
- clone the repo and open the `submission-NX-reviews.pdf` files directly.

(We tried re-typesetting with several font toolchains — none satisfied GitHub's inline
viewer, so the source PDFs are left in their original form.)
